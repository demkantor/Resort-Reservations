## Resort Reservations App
credits to John Smilga

### Built with
* react
* javascript
* context api
* styled components
* contentful - as a headless cms

### Setup
* clone or download
* npm install
    * if using contentful:
    * create account at contentul.com
    * add content
    * add api keys to .env
* npm start


Try it out here! [Beach Resort](https://beach-hotel-resort-reservations.netlify.app/).
